ratchet-rails
-------------

Build mobile app prototypes with [Ratchet](https://github.com/maker/ratchet) + Ruby on Rails.

#### Features

  * HAML Templates
  * SCSS Syntax (with helpful mixins)
  * UI Component Helper Methods
